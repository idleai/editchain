# editchain

CRDT-based edit chain experiment.

```text
crates/
  editchain-core/    # no_std schema, IDs, merge, canonical reducers
  editchain-codec/   # postcard binary frames + EC02 page format
  editchain-sync/    # sync state machine (no IO/runtime)
  editchain-node/    # segment files, CLI, JSON export
```

## Quick start

```sh
cargo build
cargo test
cargo run --bin editchain -- init my-chain
cargo run --bin editchain -- append '{"id":{"node":1,"boot":0,"seq":1},"parents":"None","actor":1,"clock":{"UnixMs":100},"scope":"None","tags":8,"kind":{"Message":{"content":{"Inline":[104,105]},"content_type":"Empty"}}}' my-chain
cargo run --bin editchain -- dump my-chain
```
